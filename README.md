## Available Feature

### `Auth`

Login and logout to your account spotify

### `Search Track`

Search and show track that you wanted

### `Recent Search`

Show last track that you has been searched

### `Create Playlist`

Creating new playlist for the tracks that you has been selected

### `Search Page`
